# AI Interview Confidence Coach 🚀

An AI-powered interview preparation platform designed to help candidates build confidence through realistic role-based simulations, real-time feedback, and performance tracking.

Demo link: [InterviewCoach](https://ai-interview-coach-pink-three.vercel.app/)

---

## ✨ Features

- **Role-Based Simulations**: Tailored interview questions for Software Development, Data Analytics, Cloud & DevOps, Cybersecurity, and more.
- **AI Chatbot Support**: Real-time assistance for app navigation and interview procedures.
- **Performance Analytics**: Track your progress with technical scores, communication metrics, and confidence levels.
- **Voice-to-Text Integration**: Realistic interview experience using STT technology.
- **Daily Challenges**: Keep your skills sharp with daily curated interview questions.
- **Secure Authentication**: Personalized user profiles with secure login and progress synchronization.

## 🛠️ Technology Stack

- **Frontend**: React, Vite, Framer Motion, Lucide React, TailwindCSS/Vanilla CSS.
- **Backend**: FastAPI (Python), Uvicorn.
- **AI/LLM**: Groq (Llama 3.1) for intelligent evaluation and chat.
- **Database/Auth**: Firebase Firestore & Firebase Auth.
- **Deployment**: Vercel (Frontend), Render (Backend).

## 🚀 Getting Started

### Prerequisites
- Python 3.9+
- Node.js 18+

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Sm-LumeCode/ai-interview-confidence-coach.git
   cd ai-interview-confidence-coach
   ```

2. **Backend Setup**:
   ```bash
   cd backend
   pip install -r requirements.txt
   # Set up your .env file with GROQ_API_KEY and Firebase credentials
   python main.py
   ```

3. **Frontend Setup**:
   ```bash
   cd frontend
   npm install
   # Set up VITE_API_URL in your .env
   npm run dev
   ```
